# Cypress Testing


## Install
```sh
# Install Cypress
npm install cypress --save-dev

# Install base package
npm install dotenv express-jwt jwks-rsa
```

## Open Cypress Launchpad
```sh
# Using npx
npx cypress open

# Using npm scripts
npm run cypress:open
```
